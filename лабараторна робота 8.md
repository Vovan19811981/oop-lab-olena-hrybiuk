# Лабораторна робота №8 — Віртуальні функції і поліморфізм

> **Мета роботи:** вивчення віртуальних функцій та типів поліморфізму в C++.

---

## Теоретична база

| Поняття | Опис |
|---|---|
| **Статичний поліморфізм** | Перевантаження функцій і операцій, шаблони — вирішується під час **компіляції** |
| **Динамічний поліморфізм** | Перевизначення `virtual`-методів — вирішується під час **виконання** |
| **Раннє зв'язування** | Виклик через об'єкт — компілятор знає метод заздалегідь |
| **Пізнє зв'язування** | Виклик через вказівник/посилання на базовий клас → обирається метод **нащадка** |
| **Абстрактний клас** | Містить хоча б один чистий ВМ (`= 0`); не можна створити об'єкт |
| **Чиста ВМ** | `virtual T f() = 0;` — реалізація обов'язкова у нащадку |
| **Віртуальний деструктор** | Забезпечує коректне знищення об'єктів нащадків через вказівник на базовий клас |

## Завдання і формули

### Завдання 1 — Площа геометричних фігур
Абстрактний клас `Figure` з методом `area()`.

| Клас | Формула |
|---|---|
| `Rectangle` | `S = a × b` |
| `Circle` | `S = π × r²` |
| `RightTriangle` | `S = ½ × a × b` |
| `Trapezoid` | `S = ½ × (a + b) × h` |

### Завдання 2 — Норма об'єктів
Абстрактний клас `Norm` з методом `norm()`.

| Клас | Формула |
|---|---|
| `Complex` | `\|z\| = √(re² + im²)` |
| `Vector` | `‖v‖ = √(Σ xᵢ²)` |
| `Matrix` | `‖A‖ = max\|aᵢⱼ\|` |

### Завдання 3 — Криві: y(x)
Абстрактний клас `Curve` з методом `compute(x)`.

| Клас | Рівняння |
|---|---|
| `Line` | `y = ax + b` |
| `Ellipse` | `x²/a² + y²/b² = 1` → `y = b√(1 - x²/a²)` |
| `Hyperbola` | `x²/a² - y²/b² = 1` → `y = b√(x²/a² - 1)` |

### Завдання 4 — Сума прогресій
| Клас | Елемент | Сума |
|---|---|---|
| `ArithmeticProgression` | `aⱼ = a₀ + j·d` | `Sₙ = (n+1)(a₀+aₙ)/2` |
| `GeometricProgression` | `aⱼ = a₀·rʲ` | `Sₙ = (a₀ - aₙ·r)/(1 - r)` |

### Завдання 5 — Стек і Черга
Базовий клас `List` з методами `push()` та `pop()`. LIFO vs FIFO.

### Завдання 6 — Площа і периметр
| Клас | Площа | Периметр |
|---|---|---|
| `Circle` | `S = πr²` | `L = 2πr` |
| `Rectangle` | `S = ab` | `L = 2(a+b)` |
| `Trapezoid` | `S = (a+b)h/2` | `L = a+b+c+d` |

### Завдання 7 — Нарахування зарплати
- `HourlyEmployee` — погодинна ставка × години (понаднормові ×1.5)
- `SalariedEmployee` — фіксований місячний оклад

### Завдання 8 — Площа поверхні тіл
| Клас | Формула |
|---|---|
| `Parallelepiped` | `S = 2(ab + bc + ca)` |
| `Tetrahedron` | `S = a²√3` |
| `Sphere` | `S = 4πr²` |

### Завдання 9 — Ссавці
Ієрархія: `Mammal` → `Human`, `Animal` → `Horse`, `Cow`

### Завдання 10 — Батько і Дитина
Клас `Father` з іменем → клас `Child` успадковує поле по батькові.

### Завдання 12 — Корені рівнянь
- `LinearEquation` — `ax + b = 0` → `x = -b/a`
- `QuadraticEquation` — `ax² + bx + c = 0`, дискримінант D

### Завдання 13 — Фігури на екрані
Абстрактний клас із: координатами центра, кутом повороту, масштабом.  
Методи: `draw()`, `hide()`, `rotate(deg)`, `move(dx, dy)`.
